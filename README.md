## How to use d.ts

0. `npm install typings -g`  
1. `typings install dt~jquery --save --global`
2. Add a comment to the top of the file: `/// <reference path="typings/globals/jquery/index.d.ts" />`

For more d.ts files? Please use `typings search`

Enjoy coding.
